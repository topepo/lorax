# as.party.C5.0 validates tree parameter

    Code
      as.party(c5_model, tree = 0)
    Condition
      Error in `as.party()`:
      ! `tree` must be >= 1, not 0.

---

    Code
      as.party(c5_model, tree = 5)
    Condition
      Warning:
      `tree` = 5 exceeds number of trials (1). Using `tree` = 1 instead.
    Output
      [1] root
      |   [2] flipper_length_mm < 206: *
      |   [3] flipper_length_mm >= 206
      |   |   [4] bill_depth_mm < 17.6: *

      |   |   |   [6] bill_length_mm < 46.5: *
      |   |   |   [7] bill_length_mm >= 46.5: *

    Code
      as.party(c5_model, tree = c(1, 2))
    Condition

      as.party(c5_model, tree = "1")

# as.party.C5.0 rejects rule-based models

    Code
      as.party(c5_rules, tree = 1)
    Condition
      Error in `as.party()`:
      ! C50 rule-based models are not supported. Use tree-based models only.
